## Roles in the Concept

- **Generic Object:** beged (בֶּגֶד) — the garment itself, any type

- **Positive Action:** labash (לָבַשׁ) — putting on, investing with garments
- **Destructive Action:** qara (קָרַע) — tearing/rending garments in grief
- **Purifying Action:** kabac (כָּבַס) — washing garments to restore purity
- **Condition:** tame (טָמֵא) — the unclean state that necessitates garment washing
