# WG: Guilt Offering

## Overview
The guilt/trespass offering concept clusters around the אשׁם root and its interaction with the broader sacrificial and sin vocabulary. The ʼasham (H817) uniquely combines the abstract guilt-state with the concrete offering that resolves it — a single word denoting both the debt and the payment. This group maps the words that define what guilt is, how it is incurred, and how it is ritually resolved.
