# WG: Cleansing / Purity

## Overview
The clean/unclean concept in Hebrew Scripture forms a fundamental binary system governing access to sacred space, fellowship, and worship. The taher/tame word families are exact antonyms, each with verb, adjective, and noun forms. Physical washing actions (kabas, rachats) serve as the ritual mechanism for transitioning from unclean to clean status.
