## Relationships

### Shared Roots
- G225 (alētheia), G227 (alēthēs), G228 (alēthinos), G230 (alēthōs) — all from alpha-privative + λανθάνω (lanthano, "to escape notice"). Core idea: "unhidden, unconcealed."
- G226 ἀληθεύω (alētheuō, "to speak the truth," 2x) is also in this root family but too rare to include as a core member.
