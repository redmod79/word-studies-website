## Continuity and Shifts

**What stays the same across testaments:**
- Cloud = visible divine presence; theophany marker
- Cloud at covenant / law-giving moments (Noah → Sinai → Transfiguration)
- Cloud as medium of divine voice
- Cloud associated with glory (kabod / doxa)

**What shifts:**
- OT: God descends into cloud to dwell with Israel and command them; NT: Christ ascends into cloud, then returns via cloud — the direction inverts
- OT: cloud primarily guides the living community; NT: cloud is the vehicle of the risen, exalted Christ
- The eschatological judgment-cloud of the prophets (Joel, Nah, Ezek) reaches its NT fulfillment in the "coming with clouds" motif (Rev 1:7)
