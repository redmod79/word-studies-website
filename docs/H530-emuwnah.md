## Lexicon Definition

Feminine of H529 (ʼêmûwn); literally firmness; figuratively security; morally fidelity: faith(-ful, -ly, -ness, -man), set office, stability, steady, truly, truth, verily.

The split between pistis and aletheia in the LXX reflects emunah's dual semantic range: when it means "faithfulness/fidelity," the LXX uses pistis; when it means "truth/reliability," the LXX uses aletheia. The Hab 2:4 rendering is pivotal: the LXX has ek pisteōs mou ("by my faithfulness"), which Paul reads as ek pisteōs ("by faith") in Rom 1:17, Gal 3:11.
