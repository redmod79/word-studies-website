# H1419 — גָּדוֹל (gāḏôl / gâdôwl)

- **Hapax Legomenon:** No

## Lexicon Definition

From H1431 (shortened); great (in any sense); hence, older; also insolent:— aloud, elder(-est), exceeding(-ly), far, (man of) great (man, matter, thing, -er, -ness), high, long, loud, mighty, more, much, noble, proud thing, sore, (very).

The term spans three conceptual poles: **quantitative** (size, magnitude, volume), **qualitative** (dignity, importance, status), and **intensive** (loud, exceeding, vehement). All three are live options in any given context.
